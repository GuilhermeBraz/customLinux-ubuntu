## Sprints Formula

<p style="text-align: center; font-size:x-large"><b>d = vT= DLYnt</b></p>

> - d = distance
> - v = velocity
> - T = time
> - D = direction (values)
> - L = stride length (deliberate practice)
> - Y = rhythm
> - n = sprints per day
> - t = length of sprint
